Feedback Page


Technical Description

To impliment the feedback page I used HTML, CSS and JavaScript. Used the form element in HTML. I catogarized the field sets in the form using attributes like fieldset and legends. Used text inputs, textareas, radio buttons, check boxes, popup lists,and file uploading fields to collect data from the user. Used different input types like texts, telephone numbers and addresses and add place holders as nedded. Used a star button rating system to rate the website. Used a reset button to reset the field and added a submit button to submit the entered data. Also marked the compulsary fields as required and used JavaScript to show the customized error massages to those fields. Also used JavaScript to show the remaining word count for textareas. Used CSS to make the web page more user friedly by adding borders, margins, background colors, font colors and paddings as needed. Also used the hover effect to make submit and reset buttons, star rating and file uploading button more visually engaging.


Accessibility

Used semantic elements like fieldset and legent to understantd the structure of the form. Used labels and types, id and classes to better accessiblity. Added place holders. Highlighted the accessing data field. Highlighting the submit and reset buttons and rating stars using hover effect. Have customized error massages for the required fields.

Team Page


Technical Description

To implement this page HTML, and CSS were used. Added a picture of the each member by horizontally aliging them. Also used the attribute alt to show the image name for the screen reader. Used CSS to show the detail box once the user clicked on a image of a team member. Also used tabindex='0' to navigate through the keyboard. Used hover effect to make detail box and the images more visually engaging. Added name, role, implimented pages and linkedin profile link of the each member to the detail box.


Accessibility

Used alt to name the images. Use flexboxes to arrange images and detail boxes. Used tabindex='0' to easily navigate though detail boxes using keyboard. Also used hover effect to make images and detail boxes more visually apealing.
